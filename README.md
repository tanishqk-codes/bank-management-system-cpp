# Bank Management System (C++)

A console-based Bank Management System in C++ with account management, transactions, file handling, and statistics features.

## Features

- Login System
- Create Account
- Show All Accounts
- Search Account
- Search Account By Name
- Update Account
- Delete Account
- Deposit Money
- Withdraw Money
- Transfer Money
- Generate Bank Statement
- Transaction History
- Bank Statistics
- File Handling
